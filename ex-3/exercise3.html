Front-end technologies are the visual aspects of a website or mobile app that users can see and interact with, such as buttons, images, and navigation. Front-end developers are responsible for the look and feel of a website, including how it looks on different devices. The three main front-end coding languages are: HTML: Hypertext Markup Language, which creates the code version of a wireframe CSS: Cascading Style Sheets, which gives HTML visual appeal JavaScript: A runtime language for web browsers that can perform actions and make decisions
